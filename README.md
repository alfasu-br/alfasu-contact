# Links Alfasu

Pagina estatica com os links oficiais da Alfasu para uso em GitHub Pages.

## Conteudo

- Site de solucoes
- Telefone
- WhatsApp
- E-mail
- Instagram
- LinkedIn
- Localizacao no Google Maps

## Como visualizar localmente

Abra o arquivo `index.html` diretamente no navegador ou rode um servidor local:

```powershell
python -m http.server 8000
```

Depois acesse:

```text
http://localhost:8000
```

## Publicacao no GitHub Pages

No GitHub, acesse:

```text
Settings > Pages
```

Configure:

- Source: `Deploy from a branch`
- Branch: `main` ou `master`
- Folder: `/root`

Depois salve e aguarde o link publico ser gerado.
